## 题目
```
已知两个有序数组, 现在要求将两个有序数组合并为一个有序数组.
```

## 信息
```
1. 两个有序数组是否都为升序 or 降序: 需要统一
2. 目标数组是升序 or 降序
```

## 代码实现

```python
# coding: utf-8

def merge_array(num1, num2):
    i, j, _res = 0, 0, list()
    len1, len2 = len(num1), len(num2)

    while i < len1 and j < len2:
        if num1[i] < num2[j]:
            _res.append(num1[i])
            i += 1
        else:
            _res.append(num2[j])
            j += 1


    if i == len1:
        _res.extend(num2[j:])
    else:
        _res.extend(num1[i:])

    return _res


if __name__ == "__main__":
    num1 = [0, 2, 4, 6, 8]
    num2 = [1, 3, 5, 7, 9, 11]
    print(merge_array(num1, num2))
```
